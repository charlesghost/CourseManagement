# Course Management 


### Running the Project

    > rails server 
    > Go on your local browser and type ---- localhost:3000 

### Database creation

    > rails bd:create
    > rails db:migrate


### Gems to install

    > gem 'rails'
    > gem 'json'
    > gem 'httparty'
    
### Relationships Strcuture

    > Stream has many disciplines
    > Discipline has many cohorts
    > Discipline belongs to stream
    > Cohort belongs to discipline
